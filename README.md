# 探秘与解构：科学视角下的玄学世界观

这是一个使用 [Slidev](https://sli.dev) 创建的演示文稿项目，主题为"科学视角下的玄学世界观——从八字命理到六爻占卜的原理与应用"。

## 主题

本项目使用 [Apple Basic Theme](https://github.com/slidevjs/themes/tree/main/packages/theme-apple-basic) 主题，提供简洁优雅的视觉效果。

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 开发模式

启动本地开发服务器，支持热重载：

```bash
pnpm dev
```

访问 http://localhost:3030 查看演示文稿

### 构建生产版本

构建用于生产环境的静态文件：

```bash
pnpm build
```

### 导出为 PDF

将演示文稿导出为 PDF 文件：

```bash
pnpm export
```

### 第一部分：溯源与定义
- 道家 vs 道教

- 道家五术概览

### 第二部分：命 — 八字命理

### 第三部分：卜 — 六爻占卜
- 六爻的原理（五种解释框架）

### 第四部分：避坑指南与总结

- 科学的玄学观

## 演讲技巧

1. **互动环节**：在讲解日元特征时，可以邀请观众分享自己的日元并验证性格特征
2. **故事案例**：使用"铜山西崩"等生动案例增强理解
3. **思考引导**：在讲到"拉普拉斯妖"时，引发观众对宇宙本质的思考
